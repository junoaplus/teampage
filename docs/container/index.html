CONTAINER 개요 컨테이너란? 호스트 OS 위에 여러 개의 격리된 환경을 만들어서 각각의 환경에서 독립적으로 프로세스와 파일 시스템 등을 실행하는 기술이다. 컨테이너 이미지란? 컨테이너 이미지는 도커와 같은 컨테이너 가상화 기술에서 사용되는 실행 가능한 소프트웨어 패키지이다. 이 패키지는 애플리케이션을 실행하는 데 필요한 모든 소프트웨어를 포함하며, 애플리케이션의 코드, 라이브러리, 실행 환경, 시스템 도구 등이 포함될 수 있다. 컨테이너 이미지는 일반적으로 컨테이너 레지스트리에서 관리되며, 필요에 따라 다운로드하거나 업로드하여 사용할 수 있다. 컨테이너 이미지를 기반으로 컨테이너를 생성하고 실행하면, 해당 컨테이너는 이미지에 포함된 애플리케이션을 실행하게 됨.
